## 🚀 Featured Project

### [llm-wiki](https://github.com/xiaobai-agent/llm-wiki) — Personal Knowledge Base Starter Kit

> **An open-source, schema-first knowledge base that makes your AI agent smarter over time.**
> Clone it, paste the schema into Claude/ChatGPT/Gemini, start building. 60 seconds to launch.

- 🧠 Three-layer architecture (Raw → Pages → Meta) with self-evolution loop
- 📖 Full design documentation ([English](https://github.com/xiaobai-agent/llm-wiki/blob/main/docs/design-document.md) | [中文](https://github.com/xiaobai-agent/llm-wiki/blob/main/docs/design-document-zh.md))
